## COVID-19 Exploratory Data Analysis (EDA)
### Data Cleaning and Preprocessing
The first major process performed in this EDA project was data cleaning and preprocessing. This involved checking the dataset for missing values, duplicate records, incorrect data types, and inconsistent data entries. Missing values were handled using suitable techniques such as filling null values, while duplicate rows were removed to improve the quality and accuracy of the dataset. Proper preprocessing ensured that the data was clean, reliable, and ready for further analysis and visualization.
### Outlier Detection & Feature Engineering
Outlier detection was another important process performed during the EDA. Boxplots and the IQR (Interquartile Range) method were used to identify unusually high or extreme values in confirmed cases, deaths, and recoveries. Feature engineering was performed to create new useful columns from the existing dataset for better analysis and interpretation. New features such as Active Cases, Death Rate, and Recovery Rate were calculated using confirmed, death, and recovered cases. These newly created features helped provide deeper insights into the severity and recovery trends of COVID-19 across different countries and improved the overall analytical understanding of the dataset.
### Statistical Analysis
Statistical analysis techniques such as mean, median, standard deviation, skewness, and correlation analysis were also applied to understand the overall behavior, variability, and distribution of the dataset. This process helped identify heavily affected countries and unusual data patterns.
### Univariate, Bivariate, and Multivariate Analysis
Different types of statistical analysis were conducted to study the dataset from multiple perspectives. Univariate analysis was used to analyze single variables such as confirmed or death cases individually to understand their distribution and spread. Bivariate analysis was performed to study relationships between two variables, such as confirmed cases and deaths, using scatter plots and correlation analysis. Multivariate analysis was used to analyze multiple variables together and identify patterns and relationships among confirmed, recovered, deaths, and active cases through heatmaps and correlation matrices.
### Data Visualization and Insight Extraction
Various data visualization techniques were used to present the analysis results in a clear and understandable manner. Bar charts, histograms, scatter plots, pie charts, boxplots, KDE plots, and correlation heatmaps were created to visualize country-wise COVID-19 statistics and relationships between variables. These visualizations helped identify important patterns, trends, and comparisons in the dataset, making it easier to extract meaningful insights about the global impact of the COVID-19 pandemic.
